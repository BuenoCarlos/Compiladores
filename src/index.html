Informações sobre a gramática

Gramática:

S ::= aA | bB

A ::= cC | d

B ::= eA | f

C ::= g | ε

First:

S = a, b

A = c, d

B = e, f

C = g, ε

Follow:

S = $

A = $

B = $

C = $

Tabela de Parsing

a b c d e f g $
S S → aA S → bB
A A → cC A → d
B B → eA B → f
C C → g C → ε
